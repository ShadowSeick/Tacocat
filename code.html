                    //Controller Function
                    function startPalindrome() {
                        
                        let textToCheck = document.getElementById("stringCheckPalindrome").value;
                        
                        if (textToCheck) {
                            let isPalindrome = checkPalindrome(textToCheck);
                            displayResultPalindrome(isPalindrome, textToCheck);
                        }
                        else {
                            alert("Enter a String!")
                        }
                        
                    }

                    //Reverse string
                    function reverseString(text) {
                        return text.split("").reverse().join("");
                    }

                    //Logic Function
                    function checkPalindrome(text) {

                        let onlyLettersText = text.replace(/[^a-zA-Z]/g, "");
                        return reverseString(onlyLettersText) == text;
                        
                    }

                    //DisplayFunction
                    function displayResultPalindrome(isPalindrome, text) {

                        let placeToShowResult = document.getElementById("result");
                        placeToShowResult.innerHTML = "";
                        
                        //Find and remove result element color class 
                        let alertClassColorMatch = /(?:alert-[\w]+)$/;
                        let classesInAlert = placeToShowResult.className;
                        let alertClassColor = classesInAlert.match(alertClassColorMatch);
                        placeToShowResult.classList.remove(alertClassColor);

                        let headingResultText = "";

                        if (isPalindrome) {
                            placeToShowResult.classList.add("alert-success");
                            headingResultText = "your text is a palindrome!";
                        }
                        else {
                            placeToShowResult.classList.add("alert-warning");
                            headingResultText = "no palindrome here!";
                        }

                        placeToShowResult.innerHTML += `<h4 class="alert-heading text-uppercase">${headingResultText}</h4> <hr>
                                                        <p class="mb-0">Your text reversed is: ${reverseString(text)}</p>`
                        placeToShowResult.classList.remove("d-none");
                    }

The Code is structured in 4 functions

TACOCAT

Taco Cat is a palindrome. A Palindrome is a word, phrase, or sequence that reads the sme backward as forward. Write the checks if the phrase entered is a palindrome.

startPalindrome()

This function controls and starts the application. It has been an iterative process to arrive here. The problem was how to "validate" the user text. The only case where the text is not valid is when it is an empty string. First I had a function which returned a boolean to check if it is a valid statement. But, the solution usually is very simple. If the string is empty its boolean value is negative, otherwise it is true. So I have used its value directly in the if/else statement.

reverseString(text)

I have made a function only for the purpose of reverse a string, because I use it in two separate functions to DRY.

checkPalindrome(text)

Here I have used a RegExp to replace any character which is not a letter. The user, intentionally or not, may choose to write a sentence with non valid characters. Before it is checked, the function will trim any unnecessary characters. Then It returns true if It is a palindrome or false if It is not.

displayResultPalindrome(isPalindrome, text)

The main purpose of this function is to display in the page the result. First we reset the alert element. Then we find and remove the result element color class. When it's first launched it won't have any color class, but the following times it will. If the function don't remove the colour classes, it won't show the colour it has to show. I have used another RegExp, but in this case it searches any alphanumerical repetition following alert-*.
next part of the function is for choosing what to display, depending if the text is a palindrome, and making visible the element and writing what it has to display.

* In this case it has been fairly easy to check for the colour classes. If I had more classes with the same structure as the colour ones, I would have to change the way the function finds the classes.
